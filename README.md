# 🌿 Know the Plant

Know the Plant is a simple web application built using **HTML, CSS, and JavaScript**. It displays information about different plants in a gallery and allows users to find their birth month plant.

## Features

- Plant image gallery
- Information about 12 plants
- Clickable images for navigation
- Birth month plant finder
- Simple dark-themed interface

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open `demo1.html` in your web browser.
